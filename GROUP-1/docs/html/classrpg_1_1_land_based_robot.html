Detailed Description

This is an abstract class containing attributes and methods for landbased navigation and handling of objects.

Parameters
name_Name of the robot.
speed_Driving speed of the robot.
width_Width of the base of the robot.
length_Length of the base of the robot.
height_Height of the base of the robot.
capacity_Payload of the arm.
intx_: X coordinate of the robot in the maze.
inty_: Y coordinate of the robot in the maze.
